Overview

The vaadin-grid element is a free, flexible and high-quality Web Component for showing large amounts of tabular data.

Features:

  • Lazy-loading
  • Virtual scrolling
  • Frozen/fixed columns
  • Customizable headers and footers
  • Custom cell renderers
  • Touch support
  • Keyboard navigation
  • Sorting
  • Accessibility
  • Declarative and imperative APIs

Getting started

Importing vaadin-grid.

Like any other web component, the normal way is to add the element to your project using bower . 

       $ bower install --save vaadin-grid

Then import the web component in your page.

      <link rel="import" href="bower_components/vaadin-grid/vaadin-grid.html" >

Instead of installing the component in your server, you might want to import the component directly from our CDN infrastruncture:

      <link rel="import" href="https://cdn.vaadin.com/vaadin-core-elements/latest/vaadin-grid/vaadin-grid.html" >

Basic usage.

Configure the element just using declarative HTML tags and attributes and add your data array via the items properties.

NameSurnameNumber
var grid = grid || document.querySelector('vaadin-grid'); HTMLImports.whenReady(function() { grid.items = [ ['Luz', 'Rojas', '+34 666 111 222'], ['Debora', 'Cabezas', '+00 987 456 765'], ['Yola', 'Prieto', '+32 123 456 789'] ]; });
